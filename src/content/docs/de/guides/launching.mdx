Dieser Leitfaden hilft Ihnen beim Starten von SAP&reg; auf Linux nach einer erfolgreichen [Installation von SAP&reg; auf Linux](../installation/).

## Startanweisungen
### Schritt 1: Startskript erstellen (Skript von Linus)
```bash
#!/bin/bash

cd ~/.wine/drive_c/users/$USER/AppData/Roaming/Zebedee/
wine zebedee.exe -f client.zbd -d &
PID=$!
wine ~/.wine/drive_c/Program\ Files/SAP/FrontEnd/SAPGUI/saplogon.exe
kill -9 $PID
```
Schreiben Sie dies in eine Datei namens `startsap` mittels eines Editors wie `vim` oder `nano` und machen Sie sie mit `chmod +x startsap` ausführbar.
Sollten Sie einen eigenen wineprefix verwenden, müssen Sie den Pfad entsprechend anpassen.

### Schritt 2: SAP&reg; starten
```bash
./startsap
```

### Schritt 3 (Optional): Verschieben des Skripts in einen bequemeren Ort
```bash
sudo cp ./startsap /usr/local/bin/startsap
```

Wenn Sie alles richtig gemacht haben, sollten Sie jetzt in der Lage sein, SAP&reg; auf Linux zu starten und zu verwenden.  
Wenn Sie Probleme haben, lesen Sie bitte die [Installationsanleitung](../installation/) oder kontaktieren Sie uns über ein [Github Issue](https://github.com/gamersi/howtosaplinux/issues/new/choose).
